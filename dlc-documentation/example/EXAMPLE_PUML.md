# This is PUML

Use this syntaxt to generate Puml-Diagrams inline
(is not rendered by Bitbucket-Cloud tho)

```puml
@startuml
Class01 <|-- Class02
Class03 *-- Class04
Class05 o-- Class06
Class07 .. Class08
Class09 -- Class10
@enduml
```

But one can take a screenshot with the Pumel-Tool in IntelliJ IDEA
and store it inside the images section!
